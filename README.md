## Sobre
É um projeto criado para praticar lógica de programação.

Desenvolvido um jogo onde ele pergunta qual é a capital do país, e você precisa escolher uma resposta entre as quatro opções que ele mostra.

## Funcionalidades
- Escolhe o número de questões.
- Escolhe uma resposta.
- Visualiza se sua resposta foi certa ou errada.
- Mostra o resultado final com o número total de questões, total acertadas, total erradas e a porcentagem do acerto.
